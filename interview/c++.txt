1、C 和 C++ 中 struct 有什么区别？
	1、C不能定义函数，但可以定义函数指针
	2、c没有protection行为，c++默认为public
	
2、C++中的 struct 和 class 有什么区别？
	1、struct默认继承和访问权限为public，class默认继承和访问权限为private
	
3、如何判断一段程序是由C编译程序还是由C++编译程序编译的？
	1、通过预编译宏，如果#define _cpluscplus,则为c++
	
4、 C和C++有什么不同?
	1、c是结构化过程的编程语言，c++是面向对象的编程语言，侧重于类的设计而不是逻辑的设计

5、引用与指针的区别是什么？
	1、引用必须初始化，指针不必
	2、引用初始化以后不能改变引用的对象，指针可以改变所指的对象
	3、不存在指向空值的引用，但是存在指向空值的指针
	
6、static全局变量与普通的全局变量有什么区别 ? 
	1、static 全局变量的作用范围为本文件
	
7、static局部变量和普通局部变量有什么区别 ？
	1、局部变量改为静态局部变量之后改变了它的存储方式即改变了它的生命周期
	
8、某文件中定义的静态全局变量(或称静态外部变量)其作用域是 () ?
	1、本文件
	
9、C++函数中值的传递方式有哪几种？
	1、值传递、指针传递和引用传递
	
10、对于一个频繁使用的短小函数,在C语言中应用什么实现,在C++中应用什么实现？
	1、c中用宏定义
	2、c++中使用inline

11、C++中 virtual 与 inline 的含义分别是什么？
	1、基类的成员函数声明为virtual，说明它是虚函数
	2、inline与函数的定义体放在一起，使该函数称为内联
	
12、VC 中，编译工具条内的 Debug 与 Release 选项是什么含义？
	1、debug选项说明为调试版本，包含调试信息，并且不做任何优化，便于程序员调试
	2、release选项说明为发布版本，它往往进行了各种优化，代码大小和运行速度上都是最优的，以便用户很好的使用
	
13、 const  与 #define 的比较 ，const有什么优点?
	1、const常量在编译期间进行类型安全检查，#define只是简单地额字符串替换

14、有了 malloc/free 为什么还要 new/delete  ？
	1、malloc/free是申请释放内存的标准库函数，new/delete是C++对象操作的运算符，调用对象的构造/析构函数

15、如果在申请动态内存时找不到足够大的内存块，malloc 和 new 将返回 NULL 指针，宣告内存申请失败。你是怎么处理内存耗尽的？
	1、判断指针是否为NULL，如果是则马上调用return终止本函数
	2、判断指针是否为NULL，如果是则马上调用exit(1)终止整个程序的运行

16、C++是不是类型安全的？
	1、不是，两个不同类型的指针可以强制转换
	
17、const 符号常量；(1)const char *p; (2)char const *p; (3)char * const p; 说明上面三种描述的区别；
	1、p是指向const char的普通指针,指向的值不能被修改
	2、p是指向const char的普通指针,指向的值不能被修改
	3、P是指向char的const指针，指针的值不能被修改
	
18、用C++写个程序，如何判断一个操作系统是16位还是32位的
	int* p; if (sizeof(p) == 2) 16位; else if (sizeof(p) == 4) 32位
	
19、用C++写个程序，如何判断一个操作系统16位还是32位的？不能用 sizeof() 函数。
	1、int a = 65535/0xFFFF; if (a==-1) 16； else 32
	
20、多态类中的虚函数表是 Compile-Time，还是 Run-Time 时建立的？
	Complie虚函数表在Complie-Time建立, 虚函数表指针是在构造时初始化的，这是实现多态的关键
	
21、内存的分配方式有几种？
	1、静态存储区
	2、栈
	3、堆
	
22、In C++, what does "explicit" mean? what does "protected" mean?
	1、explicit修饰构造函数，表示它是显示的，避免默认类型转换
	2、protected是一个函数对一个类的函数和成员变量的访问权限的控制
	
23、所有的运算符都能重载吗？
	1、不是 .
	
24、main 函数执行以前，还会执行什么代码？
	1、全局变量的初始化，全局对象的构造函数
